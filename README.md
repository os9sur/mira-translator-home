# Mira Translator

> **Ultra-lightweight (0.3MB) | Privacy-First | Highly Intuitive Translation Tool**
>
> **超轻量级 (0.3MB) | 隐私第一 | 高度直观的翻译工具**

Mira Translator 是一款追求极致美学与性能的翻译工具。体积仅 0.3MB，坚持完全免费、无会员、无广告，且无需注册登录。

## 🌟 Core Features (核心亮点)

- **Visual Element Picker (可视化拾取)**: 
  通过直观的点击，定义网页翻译区域，实现“所见即所得”的定制。
- **Innovative Learning Flow (创新学习流)**: 
  YouTube 场景下，双击字幕单词瞬间查词并存入 Google Drive 同步的生词本。
- **High-Quality Cache (单向借阅缓存)**: 
  智能锁定高质量 AI 翻译结果，在节省 API 消耗的同时守住质量底线。
- **Privacy Centric (隐私至上)**: 
  无后端服务器，支持通过 Google Drive 私有云同步配置。
